# Wireframe de Loja de Mercado

## Visão Geral
Este projeto apresenta um wireframe de **média fidelidade** para uma loja de mercado online, com estilo **tradicional** e estrutura pensada para montagem no **Figma**. O objetivo é facilitar a prototipagem de interfaces funcionais e organizadas para e-commerce de alimentos e produtos essenciais.

## Telas Incluídas
- Página Inicial (Home)
- Página de Produto
- Carrinho de Compras
- Checkout

---

## Estrutura das Telas

### 🏠 Página Inicial
- Cabeçalho com:
  - Logo à esquerda
  - Barra de busca central
  - Ícones de conta e carrinho à direita
- Menu horizontal com categorias
- Banner principal para promoções
- Filtros laterais (categorias, preço, marcas)
- Grid de produtos:
  - Imagem
  - Nome
  - Preço
  - Botão “Adicionar ao carrinho”
- Rodapé com links institucionais e informações da loja

### 📦 Página de Produto
- Breadcrumbs (ex.: Mercado > Hortifruti > Maçã)
- Imagem grande do produto
- Informações principais:
  - Nome
  - Preço
  - Descrição curta
  - Botão “Adicionar ao carrinho”
- Descrição detalhada
- Produtos relacionados (opcional)

### 🛒 Carrinho de Compras
- Lista de produtos:
  - Miniatura
  - Nome
  - Quantidade (+ / –)
  - Preço
- Totalizador
- Botão “Finalizar compra”

### 💳 Checkout
- Formulário de endereço de entrega
- Formulário de pagamento
- Resumo do pedido
- Botão “Concluir pedido”

---

## Recomendações para Montagem no Figma
- Criar frames de **1440×1024** para versão desktop
- Utilizar **Auto Layout** para listas e grids
- Aplicar tons de cinza (200–800) para manter o estilo tradicional
- Separar cada tela em uma página ou em frames lado a lado
- Criar componentes reutilizáveis (botões, cards, cabeçalho, rodapé)
